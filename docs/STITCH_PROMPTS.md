## ITERATION NOTES

### Colors to verify in Stitch output:
- Platinum White `#F8F9FB` — backgrounds must be near-white, not pure white
- Royal Navy `#1A2851` — not dark blue, not black — specifically this navy
- Coral Reef `#FF6B4A` — warm coral, not red, not orange
- Coral must be 5–8% of composition — flag if overused

### Typography to verify:
- Headings should read as Cormorant (high-contrast serif, editorial)
- Labels/nav should read as Jost (geometric sans, uppercase, tracked)
- Body should read as Inter (neutral, readable)

### Layout to verify:
- Void space: any screen should feel 40–50% empty — flag if too dense
- No rounded corners on primary containers
- No drop shadows on images (editorial flat)
- No box borders on form fields (bottom border only)
